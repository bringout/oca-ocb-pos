# PoS HR Restaurant


This module adapts the behavior of the PoS when the pos_hr and pos_restaurant are installed.


## Installation

```bash
pip install odoo-bringout-oca-ocb-pos_hr_restaurant
```

## Dependencies

This addon depends on:
- pos_hr
- pos_restaurant
